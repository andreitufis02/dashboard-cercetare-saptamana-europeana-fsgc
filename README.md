# FSGC Dashboard de cercetare

Deschide `index.html` în browser.

Conține:
- distribuția pe cartonașe;
- distribuția pe sfere FSGC;
- distribuția estimată pe vârste;
- frecvențe lexicale;
- word cloud;
- co-ocurențe și heatmap;
- analiza sentimentului;
- treemap tematic;
- codebook;
- cod-switching româno-englez;
- mențiuni politice naționale;
- clustere strategice;
- matrice oportunitate–fricțiune;
- statusul ipotezelor;
- recomandări strategice.

Notă: graficele interactive folosesc Plotly prin CDN.
